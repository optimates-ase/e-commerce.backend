## install pipenv:
```
pip install --user pipenv
```

## install dependencies
```
pipenv install
```

## activiate virtual environment
pipenv shell

## run server
```
python manage.py runserver
```


## run app
```
python manage.py startapp <app>
```

## run tests
```
python manage.py test
```

## test app
```
python manage.py test <app>
```
